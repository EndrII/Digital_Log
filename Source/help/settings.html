Справка по DigitalLog

Окно программы настроек позволяет вам настроить следующе параметры программы:

  1. Имя пользователя — пользователь который будет работать с базой данных. Добавление пользователя происходит на сервере;

  2. Пароль аутентификации пользователя;

  3. Хост на котором расположен сервер.

  4. Порт на котором работает сервер. по умолчанию 3306.

  5. База данных по умолчанию — база данных которая будет автоматически выбраться при повторном запуске программы.

  6. Параметр автоподключения — позволяет автоматически выполнять вход и выбор базы данных при повторном запуске программы.

Вы так же может сбросить все настойки на настройки по умолчанию нажав соответствующую кнопку.
