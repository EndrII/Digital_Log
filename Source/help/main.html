Справка по DigitalLog

Окно программы DigitalLog состоит из 6 основных элементов.

  1. Главное меню — содержит в себе следующие функции программы:

    1. Подключится — данная команда позволит вам вручную подключиться к базе данных если по каким-то причинам это не удалось сделать при запуске;

    2. Менеджер групп — вызовет диалоговое окно управления группами и предметами;

    3. Базы данных — позволит вам вызвать диалоговое окно баз банных, где вы сможете переключиться на новую базу данных;

    4. Настройки — позволит вам настроить электронный журнал;

    5. Управление датами — вызовет диалоговое окно управления датами.

  2. Индикатор состояния подключения к базе данных;

  3. Консоль оповещений — будет сообщать об всех изменениях в журнале, а так же о бо всех ошибках системы;

  4. Вкладка управления посещаемостью;

  5. Вкладка управления успеваемостью;

  6. Рабочая область.
